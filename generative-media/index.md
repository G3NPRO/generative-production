# Generative Media

One of the two pillars of **Generative Production**. This pillar covers producing images, video, audio, and spatial work with generative AI tools.

If you're new here, start with [`foundations/`](../foundations/) — the shared meta-skills that make both pillars one discipline. Then come back here for media-specific concepts and craft.

## Contents

- [`concepts/`](concepts/) — model basics, modalities, and the tools-and-vendors landscape.
- [`craft/`](craft/) — media-specific technique: consistency, camera motion, temporal coherence, and knowing when not to use AI.
- [`lifecycle/`](lifecycle/) — a supplemental crosswalk mapping pages to traditional film-production stages. Generative tools are collapsing these stages; this view is a reference, not a spine.
