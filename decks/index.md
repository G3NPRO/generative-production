# Decks

The whole guide, presented as slides. The same content that lives in the wiki, re-shaped so you can teach from it or walk through it with an audience.

## Site deck

[Generative Production — the study guide, presented](site-deck.html)

The site's content is chaptered by section. Each section begins with a title slide; each page within a section is a slide underneath it. Reading order follows the sidebar.

**Navigation:**

- **→ / ←** jump horizontally between section chapters.
- **↓ / ↑** walk through the pages inside the current chapter.
- **Space** advances through everything in reading order (pages, then into the next chapter).
- **`f`** fullscreen, **`s`** speaker view, **`?`** for all shortcuts.
- **`q`** or the hamburger menu's **Exit** button returns you to the wiki.

## How this is built

One master deck file aggregates slide content for every section. Each section's title slide uses a tinted background so you can feel when you cross a chapter boundary. Pages that have more to say can be split into vertical sub-slides (press `↓`); pages that are short fit on one slide.

The wiki and the deck share the same content. The wiki is for reading; the deck is for presenting.
