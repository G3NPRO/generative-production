# Foundations (Vibe-coding)

Mental models for someone who's coded little or not at all. These pages are the on-ramp before the setup and safety material.

## Planned topics

- **What is an agent?** IDE vs chat vs autocomplete — different affordances for different jobs.
- **Where does code actually run?** Local, cloud, hybrid — and why it matters for cost and trust.
- **Terminal literacy** — `cd`, `ls`, what a shell is, why the agent drops you into one.
- **Git literacy** — why version control matters *especially* for AI-generated code.
