# Foundations

The umbrella claim of this repo is that **media generation and code generation are the same discipline**. These four pages earn that claim — they're the meta-skills every section refines in its own pages.

Read in order for a first pass; each page is short.

1. [**What is Generative Production?**](what-is-generative-production.md) — the umbrella concept.
2. [**Prompting Principles**](prompting-principles.md) — the universal input skill.
3. [**Iteration and Selection**](iteration-and-selection.md) — working with a tool that produces candidates.
4. [**Tracking Your Work**](tracking-your-work.md) — staying reproducible.

[`generative-media/`](../generative-media/) and [`vibe-coding/`](../vibe-coding/) each have pages that refine these foundations for their specific medium. Cross-links appear wherever a refinement exists.
